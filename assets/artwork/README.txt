AUDIO AFICIONADOS — ARTWORK FOLDER
====================================

Place teacher-approved album artwork files here.

Naming convention:  artist-name-album-title.jpg
Examples:
  my-bloody-valentine-loveless.jpg
  radiohead-ok-computer.jpg
  pink-floyd-the-dark-side-of-the-moon.jpg
  nirvana-nevermind.jpg

Rules:
  - Lowercase only
  - Hyphens between words
  - No apostrophes, no punctuation, no special characters
  - Use .jpg for all files (JPEG format recommended)

The expected filename for each album is shown in the Artwork Setup page
in the Teacher Dashboard. Use 500x500 or larger square images.

After placing a file here, open the Artwork Setup page and click
"Mark Artwork Verified" for that album.
